About GSD

I hacked up Getting Shit Done to learn the IndededDb APIs. I ended up using jQuery Mobile UI libraries, to kick the tires and learn about mobile web apps.

In my infinite wisdom, I hadn't realized that no mobile browsers (including our beloved Firefox Mobile Beta) supported IndexedDb, so I also whipped up a sqlite backend.

I've got some other things I'd like to hack on like an Unhosted sync, a more Responsive Design, etc...

But, better yet, you've got ideas!!! File bugs or write up a wiki pages.

Hacker? Fork me on Github.
