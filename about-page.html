About GSD

Getting Shit Done!

GSD

This simple web application gives you Contexts and Next Action lists. Use them for good and not for evil. A Context is a place, tool, or person that you need to get a job done. A Next Action is a short note about a concerte step you need to take. Confused? Read the book.

GTD

Getting Things Done is a productivity method made famous in David Allen's book. Over the years, I've tailored this down to my own flow. I noticed I was abusing Evernote and other notebooks by mixing in my "Next Action" lists with Project materials and reference materials. Enter... GSD.

Example of Using GSD:

I want to see my Shopping list when I'm at the Grocery store, so under the Grocery context, I create:

  • Buy Field Roast
  • Buy Swiss Cheese
  • Buy Rye Bread

Privacy

All your data is stored on your local device. There is no login. Think of it as sticky notes on your monitor.

Mobile / Desktop

This app is usable on a Desktop Computer, Mobile Phone, or Tablet's web browser. On Apple mobile devices, just bookmark the Homepage and add to your home screen. You'll get the shiny launch icon, etc.

And that's it. Simple, no?

Why?

I hacked on Getting Shit Done, so I could learn the Firefox 4 IndexededDb APIs, Responsive and Mobile First Design methods, and to exercise my UX / Design skillz. I ended up using the jQuery Mobile UI libraries for the Desktop and mobile verion.

In my infinite wisdom, I hadn't realized that no mobile browsers (including our beloved Firefox Mobile Beta) supported IndexedDb, so I also whipped up a sqlite backend. Currently I don't have code for Chrome's IndexedDb, but the webkit API is slightly different and the spec is in flux, so … low priority.
